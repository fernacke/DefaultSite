O que é

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web e define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Como funciona

Basicamente, o HTML usa Marcação para colocar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML faz com que cada elemento HTML seja separado de outro em um documento por "tags", que consistem no nome do elemento entre "<" e ">". Hipertexto refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites.

Principais tags do HTML

<head>Cabeçalho do documento
<title>Título da página
<body>Corpo do documento
<header>Cabeçalho da página
<nav>Navegação
<main>Conteúdo principal
<article>Artigo
<footer>Rodapé da página
