O que é

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web e define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web. "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites.

Como funciona

O HTML usa "Marcação" para colocar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais em que cada elemento HTML é separado de outro em um documento por "tags", que consistem no nome do elemento entre "<" e ">".

O que você pode fazer com um Site

Um site é um espaço na internet que serve para armazenar e divulgar informações, como textos, imagens, vídeos e áudios. Os sites podem ser usados para divulgar empresas, produtos, serviços, ideias e conhecimentos. Também podem oferecer recursos, como transações comerciais.
