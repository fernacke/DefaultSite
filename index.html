Home

Seja bem-vindo(a)!

Um site é um conjunto de páginas de hipertexto interligadas, acessíveis por um nome de domínio comum. Essas páginas são hospedadas em servidores e visualizadas por meio de navegadores, podendo apresentar diversos tipos de conteúdo, como textos, imagens, vídeos e outros elementos multimídia.

Este site foi desenvolvido com o propósito de sintetizar, documentar e colocar em prática meus primeiros aprendizados sobre a criação de um website. Para sua construção, foram aplicados conceitos fundamentais de HTML, CSS e JavaScript, representando meu início na jornada de aprendizado em Desenvolvimento Web.

Sobre

Este é meu primeiro Website!

Ele foi construído princi­pal­mente com base nas aulas do Professor Guanabara sobre:

  • HTML5;
  • CSS3;
  • JavaScript.

Com um formato básico, foi desen­volvido utilizando HTML para a es­truturação das páginas, CSS para a esti­lização e JavaScript para implementar interatividade e funcionalidades dinâ­mi­cas.

Sei que ainda tenho um longo ca­minho a percorrer para me tornar uma boa programadora, mas estou orgulhosa do meu progresso!
